## Direction (options for the maintainer — not defects)

- **The advertised AI features don't exist as features.** The README sells
  Character Identification, Sentiment Analysis, Language Detection, and Plot
  Summaries, but the implementation is one freeform chat with three canned
  starter prompts. A spike to build structured, cached per-book analysis endpoints
  would deliver what the landing page already promises. (Grounded: stated-but-undelivered.)
- **No discovery surface despite the data model supporting it.** `subjects`/`languages`
  are denormalized and a recommender exists, but there's no browse-by-subject/genre
  view — only free-text search. Cheap given the existing cache + scoring code.
  (Grounded: adjacent-possible.)
- **English-only, no i18n** — flagged only because the maintainer's other apps are
  Arabic/English bilingual; this one has no locale scaffolding. Weakly grounded;
  noted so it's a conscious choice, not drift.
